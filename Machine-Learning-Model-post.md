### Machine Learning Model

### a.	Describe how Teachable Machine uses machine learning to come up with solutions to an arbitrary problem.

Is uses images as the data input to teach the model.  You must first label the images that will be used and it compares the images to determine which data label is being shown in the preview. 
	
### b. What application did you choose to create using Teachable Machine?

I choose to determine the pen color I was holding in the image.

### c. What problem is it intended to solve?

If I was using a blue pen or gray pen.

### d. Why did you choose to create this application?

I was curious if the teachable machine was able to tell the pen color if held it in a different location on the image, I found it the pen was not in the same location it could not tell if I was holding a blue or gray pen.
